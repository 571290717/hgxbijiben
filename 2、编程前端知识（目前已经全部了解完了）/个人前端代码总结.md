```html
HTML5 ，其基本格式如下

 <!doctype html>
 <html lang="en">
 <head>
 <meta charset="UTF-8">
 <meta name="viewport"
 content="width=device-width, user-scalable=no, initial-scale=1.0,
maximum-scale=1.0, minimum-scale=1.0">
 <meta http-equiv="X-UA-Compatible" content="ie=edge">
 <title>Document</title>
 </head>
 <body>
 
 </body>
</html>


HTML⽂本的结构包括“头”部分（head）、和“主体”部分（body）
HTML是由： 标签 和 内容 构成, 每个HTML页面都有两部分构成(head头和body体)，其最外层
是 <html>...</html> 标签包裹
HTML标签（标记）的语法是由 < 和 > 括起来。
HTML标签有两种： 双标签 ： <标签名>....</标签名> 和 单标签 ： <标签名 />
HTML标签中还可以添加属性： <标签名 属性名1=“值1” 属性名2=“值2” 属性名n=“值n”>....</
标签名>
HTML标签规范：标签名小写、属性使用双引号、标签要闭合。规范不遵守浏览器不会报错，会尽
量解析，大不了不显示效果 ^_^ 。

 <!-- 这就是唯一的一种HTML注释了 -->


<title> 定义了文档的标题 标题: 
    <title>标题</title>
    
    
<base> 定义了链接标签的默认链接地址
<base href="http://www.****.com/"target="_blank">

    
 <link> 定义了一个文档和外部资源之间的关系
<link type="text/css" rel="stylesheet" href="**.css"/>

    
 <meta>
定义了HTML文档中的元数据
设置编码、关键字、描述
<meta charset="utf-8"/>
<meta name="Keywords" content="关键字"/>
<meta name="Description" content="简介、描述" />

<script >
 JavaScript脚本程序
</script>

<style type="text/css">
css样式代码
</style>
    
  
    
 ###文本标签   
<hn>...</hn> 其中n为1--6的值。 标题标签（加粗、独⽴⾏）
<i>...</i> 斜体
<em>...</em> 强调斜体
<b>...</b> 加粗
<strong>...</strong> 强调加粗
<cite></cite> 作品的标题（引⽤）
<sub>...</sub> 下标 <sup>...</sup> 上标
<del>...</del> 删除线
    
    
<!DOCTYPE html>
<html lang="en"> <head>
 <meta charset="UTF-8">
 <title>⽂本标题示例</title>
</head> <body>

 <h1>h1标题</h1>

 <h4>h4标题</h4>

 <strong>strong: 强调加粗标签</strong><br/>
 <del>del: 删除线</del><br/>
 <u>u: 下划线</u> <br/><br/>
 ⽔分⼦：H<sub>2</sub>O <br/>
 4<sup>2</sup>=16
</body>
</html>
```
