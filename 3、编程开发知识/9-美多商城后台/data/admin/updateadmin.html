更新管理员数据

1、 获取修改管理员的详情信息

点就修改按钮时，我们需要先获取要修改的分组表详情信息

接口分析

请求方式： GET /meiduo_admin/permission/admins/(?P<pk>\d+)/

请求参数： 通过请求头传递jwt token数据。

返回数据： JSON

 {
        "id": "用户id",
        "username": "用户名",
        "mobile": "手机号",
        "email": "邮箱",
        "groups": ['用户组id', ...],
        "user_permissions": ['权限id', ...]
    }
参数 类型 是否必须 说明
username str 是 用户名
mobile str 是 手机号
email str 否 邮箱
groups list 是 用户组id
user_permissions list 是 权限id

后端实现
