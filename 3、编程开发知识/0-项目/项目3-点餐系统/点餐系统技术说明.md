点餐系统说明





### 技术

- 基于Python语言，版本：>=3.6及以上。
- 使用Django框架，版本：2.2.*的LTS版本。
- MySQL数据库
- 连接数据库：mysqlclient=1.4.6
- 图像处理： Pillow=5.0.0
- Web前端技术：HTML、CSS、JavaScript和Jquery等



pip install django==2.2.*  

mysql 5.6 及以上

pip install mysqlclient
