Python Web 框架要点

1. Web应用程序处理流程

2. Web程序框架的意义

  • 用于搭建Web应用程序
  • 免去不同Web应用相同代码部分的重复编写，只需关心Web应用核心的业务逻辑实现

3. Web应用程序的本质

  • 接收并解析HTTP请求，获取具体的请求信息
  • 处理本次HTTP请求，即完成本次请求的业务逻辑处理
  • 构造并返回处理结果——HTTP响应
